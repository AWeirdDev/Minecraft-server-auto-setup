# ngrok 使用方法
如果您有在安裝伺服器時使用 NGROK，請參照本指南使用。
## 啟動
啟動伺服器後需要另外啟動 NGROK，方法十分簡單，只要在伺服器的相同目錄中開啟`StartNgrok.bat`檔案即可。  
## 獲取 IP
您的 IP 會顯示在 NGROK 介面上，應該會類似下圖：  
而您的 IP 就會是下圖紅框中的部分，通常會類似 `0.tcp.jp.ngrok.io:12345`。
